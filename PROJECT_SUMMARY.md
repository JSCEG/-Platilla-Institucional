# Platilla Institucional (Plantilla SENER 2025)

## Visión General
Plantilla integral para documentos institucionales de la Secretaría de Energía (SENER). Combina clase LaTeX, recursos gráficos y guía web para garantizar identidad visual, consistencia y publicación profesional en informes como PLADESE o PRODESEN.

## Objetivos
- Unificar el diseño de documentos oficiales con portadas, tipografías y colores institucionales.
- Simplificar la producción de reportes técnicos con tablas profesionales, bibliografía APA y recuadros informativos.
- Brindar recursos web (HTML/CSS) y documentación para extender la identidad al entorno digital.

## Alcance Técnico
- Clase `sener2025.cls` con comandos personalizados para portadas, secciones y metadatos PDF.
- Documentos de ejemplo (`template-institucional.tex`, portadas alternativas, guías de tablas y citas).
- Recursos gráficos: tipografías Patria y Noto Sans, logotipos oficiales, imágenes de portada.
- Sitio de referencia (`index.html` + `css/style.css`) y guías (`GUIA_USO_TEMPLATE.md`, `GUIA_ESTILOS_WEB.md`).
- Configuración lista para `latexmk` con XeLaTeX/LuaLaTeX y Biber.
